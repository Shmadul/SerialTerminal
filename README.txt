# SerialTerminal
Serial Terminal for OSX
<=====@shmadul’s Serial Terminal ReadMe=====>
* Plug in the Standard USB portion of your Debug cable
* Put device into pwndfu via https://github.com/Shmadul/pwndfu
* Run "iRecovery -f /Path/2/PWNED/iBSS"
* Run "iRecovery -f /Path/2/Diags.img3"
* Then Run "iRecovery -c go"
* After a few seconds your screen should turn purple
* Plug in the Serial Portion of your Debug Cable
* Run ./Connect
* Type “menu -I” and press enter to activate menu
* Type “help” to view understood commands
* Type CTRL+A+\ followed by Y to disconnect
